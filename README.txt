Heroes - multiplatform turn-based game developed using SDL2 library

This game was developed as a final project for Bachelor's Degree in Informatics (Computer Science) with a major in computer graphics and software.

Structure:
-Audio: game soundtrack folder
-Program: folder containing the execussion file, dlls and game data files
-dependency: SDL2 library files
-Heroes2: source files and CMake for creating a solution
-images: game graphics folder
